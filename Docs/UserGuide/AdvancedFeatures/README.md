# Advanced Features

This section covers advanced functionality for power users who want to customize and optimize their Sharp Bridge experience.

## Available Features

### [Transformations](Transformations.md)
Create custom parameter mappings with mathematical expressions, interpolation methods, and advanced techniques.

### [Keyboard Shortcuts](KeyboardShortcuts.md)
Efficient navigation and control with context-aware keyboard shortcuts.

### [External Editor](ExternalEditor.md)
Seamless configuration editing with your preferred text editor and hot-reload capabilities.

### [Display Customization](DisplayCustomization.md)
Customize verbosity levels, parameter table columns, and visual display elements.

## Quick Start

1. **Learn the basics** - Ensure you're comfortable with [Getting Started](../GettingStarted.md) and [Console Modes](../ConsoleModes.md)
2. **Start with transformations** - Begin with simple parameter mappings in [Transformations](Transformations.md)
3. **Explore shortcuts** - Learn keyboard shortcuts to improve efficiency in [Keyboard Shortcuts](KeyboardShortcuts.md)
4. **Customize your setup** - Configure display settings in [Display Customization](DisplayCustomization.md) and external editor in [External Editor](ExternalEditor.md)

## Need Help?

- **Check the console** - Look for error messages and status information
- **Review logs** - Check the `logs/` directory for detailed information
- **Consult Troubleshooting** - Common problems and solutions in the [main troubleshooting guide](../Troubleshooting.md)
