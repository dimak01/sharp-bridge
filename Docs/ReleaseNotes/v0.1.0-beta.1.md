# Sharp Bridge v0.1.0-beta.1

⚠️ **This is a very early beta release for testing purposes**

## What's New
- **Initial Release**: First public beta of Sharp Bridge
- **iPhone to PC Bridge**: Real-time tracking data from iPhone VTube Studio to PC VTube Studio
- **Interactive Console**: Real-time monitoring with verbosity controls and keyboard shortcuts
- **Dynamic Configuration**: Hot-reload application settings without restart
- **Transformation Rules**: Customizable parameter mapping with JSON configuration
- **Firewall Scripts**: Automated UDP port setup for iPhone communication
- **External Editor Integration**: Open configuration files in your preferred editor

## Key Features Ready for Testing
- **Phone Client**: Receives tracking data from iPhone VTube Studio via UDP
- **Transformation Engine**: Processes tracking data according to configuration rules
- **PC Client**: Sends transformed data to PC VTube Studio via WebSocket
- **Console UI**: Real-time status display with service health monitoring
- **Configuration Management**: Consolidated config with automatic hot-reload
- **Recovery System**: Automatic detection and recovery from service failures

## Installation
Download `SharpBridge-v0.1.0-beta.1-win-x64.zip` and extract to your desired location.

## Quick Start
1. Run `SharpBridge.exe` (creates default config files)
2. Run `scripts/deployment/win-x64/setup-firewall.bat` as Administrator
3. Configure iPhone IP address in `Configs/ApplicationConfig.json`
4. Start VTube Studio on PC and iPhone
